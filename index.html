Core tension
Visible notice, narrow choice

Formal choice view

The notice is visible. The user has time to read. The user can either accept or leave. On this view, the consent process may appear procedurally sufficient.

Relational capability view

Even with time and visibility, autonomy may remain thin because the user cannot negotiate, partially refuse, or continue without accepting, while exit is burdened by network dependence.
